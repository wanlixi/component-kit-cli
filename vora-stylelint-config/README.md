# Stylelint Config of Vora

## Install

#### NPM

```shell
npm i @vora/stylelint-config -D
```

#### YARN

```shell
yarn add @vora/stylelint-config --dev
```

## Usage

```js
{
  "extends": ["@vora/stylelint-config"]
}
```
